💾 Export Backup

Download all your student and payment data as a .json file to your device. Keep this file safe — it is your complete backup.

📌 Tip: Take a backup at least once a week. Store the file in Google Drive, a USB drive, or WhatsApp "Saved Messages" for safety.

📂 Restore from Backup

Select a previously exported .json backup file to restore all data. Warning: This will replace all current data.

⚠️ Caution: Restoring will overwrite existing students and payment records. Take a backup first before restoring.

🕒 Auto-Backup History (Browser)

The system automatically saves a snapshot every time data changes. These are stored in the browser and can be restored quickly.
