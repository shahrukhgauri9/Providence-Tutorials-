# Providence Tutorials — Fee Management System

> A complete, offline-capable fee management system for coaching institutes.
> Built with pure HTML, CSS & JavaScript. No server or database required.

![Providence Tutorials](logo.svg)

## ✨ Features

- 🔐 **Secure Login** — Multi-user login with lockout protection
- 👨‍🎓 **Student Management** — Full student records with scholar numbers
- 💰 **Fee Collection** — Record payments, track dues automatically
- 🖨️ **Fee Receipts** — Print half-A4 receipts
- 📲 **WhatsApp Reminders** — One-click fee reminders to parents
- 💾 **Backup & Restore** — Export/import JSON backups
- ⚙️ **User Management** — Add staff accounts, change passwords

## 🌐 Deploying on GitHub Pages

1. Push all files to your GitHub repository
2. Go to **Settings → Pages**
3. Set Source to **Deploy from a branch → main → / (root)**
4. Save — your site will be live at `https://YOUR-USERNAME.github.io/REPO-NAME/`

## 💡 How Data is Stored

All data is stored in the **browser's localStorage** on the device being used.
- Data is private to each browser/device
- Use the **Backup & Restore** feature to export data regularly
- Import the backup on a new device to transfer data
